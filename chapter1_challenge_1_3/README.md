# Dungeon Game

A text-based adventure game where the player must survive 5 rooms in a dungeon.

## Gameplay

*   **Goal:** Survive through 5 rooms.
*   **Health:** Starts at 100. If health drops to 0 or below, the game ends.
*   **Events:** In each room, a random event occurs:
    1.  **Trap:** You lose 20 health.
    2.  **Healing Potion:** You gain 15 health (capped at 100).
    3.  **Monster:** You must guess a number between 1 and 5 to defeat the monster.

## How to Run

Run the `DungeonGame` class. Follow the on-screen prompts to play.
