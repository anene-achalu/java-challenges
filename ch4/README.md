# JavaFX Hello World (Chapter 4)

This is a standard JavaFX "Hello World" application.

## Functionality

*   **GUI:** Displays a simple window with a "Hello!" button.
*   **Interaction:** Clicking the button updates a label to say "Welcome to JavaFX Application!".
*   **Structure:** Uses FXML for the UI layout and a Controller class for logic.

## How to Run

Run the `HelloApplication` class to launch the GUI.
